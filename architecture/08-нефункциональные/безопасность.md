# Безопасность

## Где хранятся секреты
- `.env` — в .gitignore, никогда не коммитится
- `.env.example` — шаблон с пустыми значениями
- `config.py` — единственная точка чтения через pydantic-settings

## Журнал действий
- `generation_log` в SQLite: user_id, username, category, timestamp
- Логи в stdout: INFO на каждый /start и аудит
- Не храним: текст карточек, результаты аудитов (пока)

## Опасная тройка

| Угроза | Защита |
|---|---|
| Личные данные (Telegram ID) | Храним в SQLite, не передаём наружу |
| Чужой контент (карточки WB/Ozon) | Не сохраняем, только обрабатываем transient |
| Отправка наружу (AI API) | Только обезличенный текст карточки, без user_id |

## Права доступа

| Действие | Гость | Пользователь | Админ |
|---|---|---|---|
| /start | ✅ | ✅ | ✅ |
| Аудит | ❌ (whitelist) | ✅ | ✅ |
| /myid | ✅ | ✅ | ✅ |
| /stats | ❌ | ❌ | ✅ |
| Одобрение пользователей | ❌ | ❌ | ✅ |
