# Первая фича

## Проблема
Селлер создал карточку WB/Ozon, она не продаёт. Он не знает почему. Нужен инструмент, который по ссылке скажет: что не так, как исправить, с чего начать.

## Бюджет
1 неделя разработки (движок на 80% готов из Ai Маркетолог).

## Решение: базовый аудит карточки
- `/start` → бот запрашивает ссылку
- Вставить URL → бот загружает карточку → AI анализирует 5 блоков
- 5 сообщений отчёта с приоритетами 🔴🟡🟢
- Кнопка «Копировать отчёт»

## Ловушки
1. WB/Ozon блокируют парсер → fallback на ручной ввод текста
2. AI галлюцинирует → жёсткий промт с требованием обоснования
3. Отчёт слишком длинный → разбивка на 5 сообщений
4. Пользователь даёт левую ссылку → валидация URL
5. Бесплатные аудиты жгут токены → лимит 3 бесплатных

## Что НЕ делаем в этой фиче
- Генерация текста (мы не копирайтер)
- Инфографика
- Сравнение с конкурентами (v2)
- История аудитов (v2)
- Платный тариф и оплата (отдельная фича)
- Веб-интерфейс (Фаза 3)
