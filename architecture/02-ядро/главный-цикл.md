# Главный цикл аудита

```
1. /start → бот: «Отправь ссылку на карточку WB или Ozon»
2. Пользователь → URL карточки
3. Бот → «🔍 Анализирую карточку...»
4. Ядро → audit_card(url):
   a. detect_platform(url) → "wb" / "ozon"
   b. fetch_product_page(url) → HTML
   c. extract_product_text(html, platform) → текст карточки
   d. build_audit_prompt(text, platform) → промт
   e. call_ai(prompt) → JSON → AuditReport
5. Бот → 5 сообщений отчёта:
   📝 ЗАГОЛОВОК — оценка, ошибки, правки
   📸 ФОТО/ВИДЕО — оценка, что добавить
   📄 ОПИСАНИЕ — структура, болеты, LSI
   🔍 SEO — ключи, переспам, релевантность
   🕵️ КОНКУРЕНТЫ — сравнение, точки обхода
6. Каждый пункт с меткой 🔴🟡🟢
7. Кнопка [📋 Копировать отчёт]
```

## Режимы

### Базовый
Пользователь даёт ссылку → аудит по 5 блокам.

### С ручным вводом (fallback)
Если URL не загрузился → «Скопируй текст карточки вручную и отправь»

## Обработка ошибок
- Битый URL → «Не удалось загрузить. Проверьте ссылку.»
- Таймаут → то же
- WB/Ozon заблокировали → fallback на ручной ввод
- AI невалидный JSON → 1 повтор, затем raw_response
