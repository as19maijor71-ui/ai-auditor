# Путь данных

```
1. Telegram → URL (строка)
2. handlers.py → detect_platform(url) → "wb" | "ozon" | None
3. url_fetcher.py → fetch_product_page(url) → HTML (строка)
4. url_fetcher.py → extract_product_text(html, platform) → текст (строка ≤ 3000)
5. prompts.py → build_audit_prompt(text, platform) → промт (строка)
6. generator.py → call_ai(prompt) → ответ AI (строка)
7. generator.py → parse_audit_response(response) → AuditReport (Pydantic)
8. handlers.py → format_audit_messages(report) → список строк (5 блоков)
9. handlers.py → отправка 5 сообщений в Telegram
10. storage.py → log_audit(user_id, report) → запись в SQLite
```

## Защита на каждом шаге

| Шаг | Защита |
|---|---|
| URL | detect_platform валидация |
| HTTP | Таймаут 5 сек, обработка 403/404/429 |
| HTML → текст | JSON-парсинг с try/except, обрезка до 3000 |
| AI → JSON | 1 повтор при ошибке, fallback на raw_text |
| Вывод | _safe_send (чанки по 4096, HTML-escape) |
