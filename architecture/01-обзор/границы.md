# Границы системы

## Что внутри
- Telegram-бот (aiogram polling)
- URL-парсер (httpx)
- AI-генератор (OpenRouter → YandexGPT fallback)
- SQLite (FSM + whitelist + audit_log)
- Rate limiter (in-memory)

## Что снаружи
- Telegram API
- OpenRouter API
- YandexGPT API
- WB / Ozon (источник HTML)
- Пользователи (селлеры, менеджеры)

## Граница доверия
- Ввод пользователя: URL → валидируется (detect_platform), макс длина
- HTML с WB/Ozon: парсится, обрезается до 3000 символов
- Ответ AI: валидируется (JSON → AuditReport), fallback на raw
- Ничего не исполняется, не сохраняется кроме user_id для статистики
