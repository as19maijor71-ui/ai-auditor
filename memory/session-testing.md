# Старт сессии — тестирование AI-аудитора 24.05.2026

## Где мы

Бот `@card_sells_bot` развёрнут на сервере TimeWeb, работает. Прошли полный цикл разработки MVP.

## Что тестируем

### Основной сценарий (экспорт из ЛК)

1. Открой Telegram → `@card_sells_bot` → `/start`
2. Кнопка «📊 Загрузить экспорт WB/Ozon»
3. Отправь тестовый файл:
   - `test_wb_export.xlsx` (5 товаров WB: спортпит, кухня, игрушки, техника, дом)
   - `test_ozon_export.xlsx` (5 товаров Ozon: повербанк, чай, кофе, лампа, рюкзак)
4. Бот покажет список товаров. Выбери один.
5. Бот покажет чеклист: что есть в экспорте + что нужно добавить
6. Выполняй пункты по порядку:
   - Скопируй описание из карточки (Ctrl+A → Ctrl+C) → вставь
   - Скопируй характеристики → вставь
   - Пришли скриншот главной страницы
   - Пришли скриншоты каждого фото по одному
   - Опиши видео (текстом)
7. Бот отмечает выполненные пункты: ✅ / ⬜
8. В любой момент жми «Запустить аудит»
9. Получи полный отчёт: оценка, заголовок, фото, описание, SEO, конкуренты, итог

### Что проверять

| Критерий | Что смотреть |
|---|---|
| Чеклист | Пункты отмечаются ✅ при выполнении, не плодятся сообщения |
| Спам | Нет 9 одинаковых сообщений при вставке длинного текста |
| Фото | Каждый скрин подтверждается «Скриншот N получен» |
| Лимит | Не вылезает «лимит исчерпан» (ты в whitelist) |
| Аудит | Конкретные цифры: длина заголовка в символах, количество фото, ИИ-глютен |
| Оценка | На первом месте в отчёте, крупно, со звёздами |
| Конкуренты | Отдельный блок с конкретными рекомендациями |
| Кнопки | «Запустить аудит» и «В главное меню» всегда видны |
