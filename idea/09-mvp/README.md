# 09-mvp — Минимально жизнеспособный продукт

## Одна убийственная фича

**AI-аудит карточки по ссылке.** Селлер вставляет URL → AI загружает карточку → анализирует 5 блоков → выдаёт отчёт с приоритетами.

Пример вывода:

```
📝 ЗАГОЛОВОК — 65/100
🔴 Слишком длинный (87 символов). WB обрезает в выдаче.
🟡 Ключ «толстовка» повторяется 3 раза — переспам.
🟢 Добавить LSI-ключ «оверсайз с капюшоном».

📸 ФОТО/ВИДЕО — 40/100
🔴 Нет фото в использовании.
🔴 Только 3 фото из 10 возможных.
🟡 Инфографика на 1 из 3 фото — добавить на все.
... (ещё 3 блока)
🕵️ КОНКУРЕНТЫ — сравнение с топ-5
```

## Скоуп

### Входит
- [ ] Telegram-бот (aiogram, polling, SQLite)
- [ ] Приём ссылки WB/Ozon
- [ ] Парсинг HTML (__NEXT_DATA__ / ld+json)
- [ ] AI-аудит по 5 блокам
- [ ] Приоритеты: красный/жёлтый/зелёный
- [ ] 5 сообщений-отчёта
- [ ] Кнопка «Копировать отчёт»
- [ ] Rate limiting + whitelist
- [ ] Статистика (/stats)
- [ ] Docker-деплой

### НЕ входит
- [ ] Генерация текста (мы не копирайтер)
- [ ] Инфографика / картинки
- [ ] Сравнение с конкурентами (v2)
- [ ] История аудитов (v2)
- [ ] Платный тариф (v2)
- [ ] Веб-интерфейс (v3)
