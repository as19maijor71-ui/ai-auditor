# 05-principles — Принципы

## 1. Аудитор, не копирайтер
Мы не пишем новый текст. Мы анализируем существующий и находим ошибки. Не «напиши заголовок», а «твой заголовок слишком длинный, WB обрезает в выдаче — сократи до 60 символов».

## 2. Приоритеты, не список
Селлеру не нужен список из 30 проблем. Ему нужны 3-5 самых критичных с пометками: исправить сейчас, важно, желательно.

## 3. Конкретика, не общие слова
Не «улучши описание», а «добавь боли клиента в первые 3 строки: напиши не "толстовка из футера", а "замерзаешь в обычной толстовке? Эта греет при -10"».

## 4. Обоснование каждого пункта
Почему это ошибка, как это влияет на ранжирование, что будет если исправить.
