## Что украсть

- Приоритеты 🔴🟡🟢 — из Pagespeed Insights

## Чего избегать

- «Ещё один генератор текста» — это занято
- Сложный веб-интерфейс — барьер для селлеров
- Цена > 2000₽ — не зайдёт для новичков

## Дельта (уникальность)

Мы — единственный AI-аудитор карточек. Не генератор, не аналитика, не инфографика. Аудитор: нашёл ошибку → объяснил почему → дал правку → расставил приоритеты. Никто в этой нише не работает.
